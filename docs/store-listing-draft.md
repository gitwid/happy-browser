# Store Listing Draft

## Short Description

Navigate pages with arrows, gestures, and large next/previous controls.

## Detailed Description

Happy Browser makes ordinary websites easier to move through.

It adds simple previous and next controls to pages such as search results, shopping listings, galleries, archives, and paginated content. Use the left and right arrow keys, horizontal two-finger scroll gestures, or the large floating buttons.

Happy Browser analyzes each page locally to find likely navigation controls such as next, previous, and load-more buttons. When it is not confident, it avoids claiming the page is ready.

Features:

- Left/right keyboard navigation
- Horizontal gesture navigation
- Large draggable page controls
- Draggable on/off toggle
- Local detection of next, previous, and load-more controls
- Scroll fallback when a page has no reliable navigation target
- Options for visibility and debug logging

Privacy:

Happy Browser performs page analysis locally in your browser. It does not transmit browsing history, page content, clicks, form values, or navigation analysis to a server.

## Permission Explanation

Happy Browser needs access to normal web pages so it can inspect visible page controls and find previous, next, and load-more navigation targets. This analysis runs locally in the browser and is used only to provide the navigation controls shown to the user.

Storage permission is used to save extension preferences.
